## Configuration

**TypeScript (`tsconfig.json`):**
- `baseUrl`: `src` (enables bare imports from src root)
- `strictNullChecks`: true
- `noImplicitAny`: true
- `noUncheckedIndexedAccess`: true
- `isolatedModules`: true
- `module`: ESNext, `moduleResolution`: node
- `inlineSourceMap` and `inlineSources`: true
- Libs: DOM, ES5, ES6, ES7

**Environment:**
- `.env` file (gitignored) with `OBSIDIAN_DEV_VAULT_PATH` — used only by esbuild watch mode to auto-copy build artifacts to a dev vault
- `.env.example` documents the expected variable

**Build Pipeline (`esbuild.config.mjs`):**
1. Entry point: `src/main.ts`
2. Output: `main.js` (CJS format, single bundle)
3. Externals: `obsidian`, `electron`, all `@codemirror/*` and `@lezer/*` packages, Node builtins
4. Custom `css-concat` plugin: concatenates `src/styles/*.css` in defined order into root `styles.css` (and `src/styles.css` convenience copy)
5. Custom `dev-vault-copy` plugin: copies `main.js`, `manifest.json`, `styles.css` to dev vault on each rebuild (when `OBSIDIAN_DEV_VAULT_PATH` is set)
6. Production mode: minified, no sourcemap, tree-shaken, single rebuild then exit
7. Dev mode: watch with inline sourcemaps

**NPM Scripts:**
```bash
npm run build    # tsc -noEmit -skipLibCheck && node esbuild.config.mjs production
npm run dev      # node esbuild.config.mjs  (watch mode)
npm test         # vitest run
npm run lint     # eslint .
npm run version  # version-bump.mjs + git add manifest.json versions.json
```
