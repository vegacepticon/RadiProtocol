## Phase 49 ‚ÜĒ Phase 50 Conflict Scenario (labeled-body.canvas)

This fixture is the regression target for the exact scenario Phase 50.1 exists to resolve: an Answer node (`n-ans`) carries `displayLabel: "Small"`, and Phase 50's `edge-label-reconciler` would legitimately label every incoming edge of that Answer with `"Small"`. In this fixture both the Question‚ÜíAnswer edge (e4) and the Loop‚ÜíQuestion body edge (e2) carry the synced label `"Small"`. Under Phase 49 this broke LOOP-04 validation (the loop had two labeled outgoing edges ‚ÄĒ e2 body and e3 exit ‚ÄĒ misclassified as two exits). Under Phase 50.1 validation accepts because only e3 starts with `+`.

Plan 03's Runner test will assert: loop picker renders exactly two buttons ‚ÄĒ one exit (`–≤—č—Ö–ĺ–ī` after `stripExitPrefix("+–≤—č—Ö–ĺ–ī")`) and one body button (`nodeLabel(n-q1)` which renders the Question target preview, per Phase 49 D-11/D-12 carried forward).
