  <behavior>
    **performMove(srcPath, srcKind, dstFolder):**
    - If srcKind === 'file' and dirname(srcPath) === dstFolder → return (no-op)
    - If srcKind === 'folder' and (srcPath === dstFolder || dstFolder.startsWith(srcPath + '/')) → throw 'Нельзя переместить папку внутрь самой себя.'
    - file branch: moveSnippet, no canvas-sync
    - folder branch: moveFolder + rewriteCanvasRefs(Map([[toCanvasKey(old), toCanvasKey(new)]])) + expand-state prefix rewrite + Notice

    **dragstart (on any row):** sets the correct MIME with node.path, sets effectAllowed='move', adds 'is-dragging' class to row.

    **dragover (on row):**
    - Check `ev.dataTransfer.types.includes(MIME_FILE)` OR `includes(MIME_FOLDER)` — if NEITHER, return immediately (do NOT preventDefault)
    - If our MIME is present, compute target folder:
      - row is folder → target = row.path
      - row is file → target = dirname(row.path)
    - Check if drop would be valid (file already in that folder → no; folder into self/descendant → forbidden)
    - If forbidden: add `.radi-snippet-tree-drop-forbidden` class (no preventDefault — cursor stays as not-allowed)
    - If valid: `ev.preventDefault()`, `ev.dataTransfer.dropEffect = 'move'`, add `.radi-snippet-tree-drop-target` class

    **dragleave (on row):** remove both drop-target and drop-forbidden classes. Ignore if relatedTarget is inside row (child enter/leave noise).

    **drop (on row):**
    - `ev.preventDefault()`, remove all drop classes
    - Read `ev.dataTransfer.getData(MIME_FILE)` and `getData(MIME_FOLDER)`
    - Determine srcPath, srcKind
    - Compute target folder as in dragover
    - `await this.performMove(srcPath, srcKind, targetFolder)`

    **dragend:** cleanup — remove 'is-dragging' from source row and any stray drop-target/drop-forbidden from the tree.
  </behavior>
