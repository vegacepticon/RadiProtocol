### Design Notes (Non-blocking, Pre-existing)

**SEP-01 deferred pick-up:** The global separator setting is read into `ProtocolRunner` at `openCanvas()` time. A mid-session settings change silently takes no effect until the canvas is reopened. Correct by design (logged in STATE.md) but no UI hint for the user.

**EditorPanel form reads stale disk data:** `renderNodeForm()` at `editor-panel-view.ts:229` calls `vault.read()` directly rather than `getCanvasJSON()`. Form may show pre-save disk data when canvas is open with unsaved in-memory edits. Save path is correct; only the form load path is affected. Pre-existing; outside Phase 17 and v1.2 scope.

**Dead CSS:** `.rp-legend`, `.rp-legend-row`, `.rp-legend-swatch` remain in styles.css (lines 67–89). No code emits these at runtime. Cosmetic, no user impact.
