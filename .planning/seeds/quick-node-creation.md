Add buttons in the node editor sidebar for quick creation of related nodes:
- "Add question after this node" — creates a question node, positions it downstream, auto-links
- "Add answer to this question" — creates an answer node, connects to current question
- Obsidian command palette commands for the same actions

Requires research into Obsidian Canvas API capabilities for programmatic node/edge creation.
