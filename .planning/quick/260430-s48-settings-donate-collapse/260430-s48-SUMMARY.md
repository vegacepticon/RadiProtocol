# Quick 260430-s48 — Settings Donate Reorder & Collapse Summary

Two small UX tweaks to the RadiProtocol settings tab: move "Помочь разработке"
from the top of the Settings tab to the very bottom (below Storage), and hide
the four wallet addresses inside a native `<details>` disclosure so the
addresses no longer dominate vertical real estate.

## Before / After

**Before** — `display()` rendered: Donate (heading + intro + 4 wallet rows
flat in `containerEl`) → Runner → Protocol → Output → Storage.

**After** — `display()` renders: Runner → Protocol → Output → Storage →
Donate (heading + intro paragraph + collapsed `<details>` containing the 4
wallet rows). The disclosure summary reads "Показать адреса кошельков" and
is collapsed by default (no `open` attribute). Clicking the copy button on
any wallet (after expansion) still triggers `navigator.clipboard.writeText`
+ "Адрес скопирован" Notice — copy logic byte-identical.

## Changes

### `src/settings.ts`

- Moved the entire donate block (heading + intro paragraph + wallet for-loop)
  from immediately after `containerEl.empty()` to immediately before the
  closing `}` of `display()` — now renders after the Session folder Setting.
- Inside the moved block, introduced
  `const detailsEl = containerEl.createEl('details', { cls: 'rp-donate-details' })`
  with a child `<summary class="rp-donate-summary">Показать адреса кошельков</summary>`.
- Changed the wallet for-loop's `new Setting(containerEl)` to
  `new Setting(detailsEl)` so the four wallet rows are children of the
  `<details>` instead of `containerEl`.
- Updated the inline comment (was "Top-most per ROADMAP SC#1") to
  "Quick 260430-s48: relocated to bottom; addresses collapsed behind <details>".
- Runner / Protocol / Output / Storage Setting blocks are byte-identical to
  their pre-edit content — only their relative position shifted up.
- No imports changed; `DONATE_WALLETS`, `DONATE_INVITATION_TEXT`,
  `DONATE_NOTICE_TEXT`, `DONATE_TOOLTIP_TEXT` already imported.

### `src/styles/donate-section.css`

Append-only — added a Quick 260430-s48 block at the bottom under the comment
`/* Quick 260430-s48: collapsible wallets */`:

- `.rp-donate-details { margin-bottom: var(--size-4-2); }`
