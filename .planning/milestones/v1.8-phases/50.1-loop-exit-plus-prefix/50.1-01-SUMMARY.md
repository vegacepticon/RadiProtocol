# Phase 50.1 Plan 01: node-label Redefinition + stripExitPrefix + EDGE-03 Insert — Summary

One-liner: Phase 50.1 foundation plan — replaces the Phase 49 `isExitEdge = isLabeledEdge` alias with a real `+`-prefix predicate, adds `stripExitPrefix` caption extractor, inserts EDGE-03 requirement, and marks EDGE-01 as superseded. `nodeLabel` and `isLabeledEdge` stay byte-identical so the Phase 50 EdgeLabelReconciler contract is preserved.

## What Shipped

### `src/graph/node-label.ts` (modified)

- **NEW export `stripExitPrefix(label: string): string`** — D-09 body verbatim:
  ```typescript
  return label.trim().slice(1).replace(/^\s+/, '');
  ```
  Semantics: trim outer whitespace, drop exactly one `+`, strip any leading whitespace (ASCII space, nbsp, etc.) following the `+`. `"+"` and `"+ "` yield `""` (validator D-08 rejects upstream). `"++foo"` yields `"+foo"`.

- **Redefined `isExitEdge(edge: RPEdge): boolean`** — D-10 body verbatim:
  ```typescript
  return edge.label != null && edge.label.trim().startsWith('+');
  ```
  Supersedes the Phase 49 alias `export const isExitEdge = isLabeledEdge`. The two predicates are now semantically distinct:
  - `isLabeledEdge`: "has any non-empty trimmed label" — Phase 50 reconciler discriminator (preserved, byte-identical).
  - `isExitEdge`: "label starts with `+` after trim" — Phase 50.1 runtime exit dispatch + validator LOOP-04 + RunnerView caption arm.

- **Byte-identical preserved:**
  - `nodeLabel` (lines 18-29) — unchanged.
  - `isLabeledEdge` (lines 40-42) — unchanged. Phase 50 EdgeLabelReconciler uses this predicate for the Answer↔edge label sync discriminator.

- `describe('isExitEdge (D-07)')` REPLACED with `describe('isExitEdge (D-10 — edge label starts with "+" after trim)')`:

  - Dedicated `undefined` test + dedicated `null` defensive test.
  - **Alias-removal regression:** `expect(isExitEdge).not.toBe(isLabeledEdge)` guards against silent future re-alias.
- NEW `describe('stripExitPrefix (D-09 — trim, slice(1), then /^\s+/)')` block APPENDED:
  - 7-row `it.each` matrix covering all D-09 behaviours.

Tests: **40 passed / 0 failed** in `node-label.test.ts` alone (+18 net new tests vs Phase 49 baseline of 22: old isExitEdge block had 3 tests, new isExitEdge block has 13 tests, plus 7 new stripExitPrefix tests = +17 net, plus alias-removal regression = +18).
