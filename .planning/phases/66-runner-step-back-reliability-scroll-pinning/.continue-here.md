## Decisions Made

1. **Two-layer Back guard** — Runner-side `_stepBackInFlight` (Plan 01) + view-side `button.disabled = true` (Plan 03). The view-side guard is intentionally synchronous and simple; the runner-side guard catches anything that slips through.
2. **Unconditional scroll-to-bottom** — Chosen over conditional scroll preservation because the runner only re-renders on user action; user has not yet started reading mid-flow (D-11).
3. **Deep-copy loop frames in undo snapshots** — Shallow spread `[...stack]` was insufficient because `advanceOrReturnToLoop` mutates `frame.iteration`. Deep copy via `.map(f => ({ ...f }))` prevents snapshot corruption.
