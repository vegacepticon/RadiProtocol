## Текст ошибок + стрип `+`

### Error copy

| Option | Description | Selected |
|--------|-------------|----------|
| Принять как есть (Recommended) | 5 текстов зафиксированы verbatim: D-01/D-01b/D-02/D-03/D-04 (в CONTEXT.md → D-04/D-05/D-06/D-07/D-08). | ✓ |
| Переформулировать | Пользователь предложит свою версию. | |
| Объединить D-01/D-01b | Откатить решение — один общий текст. | |

**User's choice:** Принять как есть — locked as D-04/D-05/D-06/D-07/D-08 in CONTEXT.md.
**Preview accepted:**
- D-04: `Loop-узел "{label}" не имеет выхода. Пометьте ровно одно исходящее ребро префиксом "+" — текст после "+" станет подписью кнопки выхода.`
- D-05: `Loop-узел "{label}" не имеет выхода с префиксом "+". Добавьте "+" к одному из помеченных рёбер ({edgeIds}) — текст после "+" станет подписью кнопки выхода.`
- D-06: `Loop-узел "{label}" имеет несколько "+"-рёбер: {edgeIds}. Должно быть ровно одно выходное ребро — уберите "+" с остальных.`
- D-07: `Loop-узел "{label}" не имеет тела — добавьте исходящее ребро без префикса "+".`
- D-08: `Loop-узел "{label}": "+"-ребро {edgeId} не имеет подписи — добавьте текст после "+".`

### Strip rule

| Option | Description | Selected |
|--------|-------------|----------|
| Стрип ровно 1 `+` + trimStart (Recommended) | `label.trim().slice(1).replace(/^\s+/, '')`. «+ выход» → «выход»; «++foo» → «+foo»; «+» → пусто → D-08. | ✓ |
| Стрип всех ведущих `+` | `label.trim().replace(/^\++\s*/, '')`. «++foo» → «foo». Размывает концепцию «префикс». | |

**User's choice:** Стрип ровно 1 `+` — locked as D-09.
