```css
/* Phase 36: fix spacing between type label and value (CLEAN-02) */
.radi-snippet-editor-row {
  display: flex;
  flex-direction: row;
  align-items: center;
  gap: var(--size-4-2);
  margin-bottom: var(--size-4-2);
}

.radi-snippet-editor-type-static {
  font-size: var(--font-ui-small);
  color: var(--text-muted);
}

.radi-snippet-editor-type-toggle {
  display: flex;
  flex-direction: row;
  gap: var(--size-4-1);
}
```

The key fix is `gap: var(--size-4-2)` on `.radi-snippet-editor-row` which adds spacing between the `<label>Тип</label>` and the adjacent `<span>JSON</span>` (edit mode) or toggle buttons (create mode).

Note: The `.radi-type-toggle` rules at line 408-430 style the toggle BUTTONS inside the toggle container. The new `.radi-snippet-editor-type-toggle` rule styles the toggle CONTAINER itself (the wrapper div with class `radi-snippet-editor-type-toggle`). These are different elements — both rules are needed. The existing `.radi-type-toggle` at line 408 targets elements with class `radi-type-toggle`, but the modal uses class `radi-snippet-editor-type-toggle` on the wrapper.

After adding CSS, run `npm run build` to regenerate styles.css.
